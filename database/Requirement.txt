1.For insert, call insertData(Year, Month, Date, Time, Name, Price), return data No.

Year: INT NOT NULL (e.g. 2024)
Month: INT NOT NULL (e.g. 02, 11)
Date: INT NOT NULL (e.g. 01, 31)
Time: INT NOT NULL (from 00:00 to 23:59)(e.g. 1055 for 10:55, 2048 for 20:48)
Name: VARCHAR(20) NOT NULL (1~20 chars)
Price: INT NOT NULL

Add tags, call insertTag(Number, Tag), return 1 for success.

Number: data number from "insertData"
Tag: VARCHAR(20) NOT NULL (1~20 chars)

2.For searching, call

searchYear(Year, mode) (mode can be GT, EQ, LT)(>, =, <)
searchMonth(Month, mode)
searchDate(Date, mode)
searchTime(Time, mode)
searchName(Name)
searchPrice(Price, mode)
searchNo(No, mode)

Return a list of item's all data.
